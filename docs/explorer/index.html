Paper Structure Knowledge Graph

A FAIR, ontology-grounded model of a scientific paper's structure — section hierarchy, every element anchored to its page region, full processing provenance. Reference layer for evaluating AI pipelines on document structuring.

Priscila Saboia Moreira · DAC DOME (Data, Operations, Modeling, and Engineering), University of Notre Dame

Containment uses po:contains — the property DoCO itself is built on. Sections nest recursively (subsection ⊂ section). A figure and its caption are grouped in a doco:FigureBox, the DoCO-blessed pattern. Every leaf element also carries pax:onPage and a pax:region → oa:ResourceSelection whose oa:FragmentSelector uses W3C Media Fragments (xywh=…) against the page image — a compact application profile of Web Annotation, deliberately not a conforming oa:Annotation.
